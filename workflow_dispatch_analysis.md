## 2. Cascade Flow Sequences

When a core package is updated, it triggers a chain reaction of automated updates. Below are the execution chains:

### Chain A: Core Pattern Collector Cascade
1. **[pattern-collector](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector)** is published.
2. Triggers dispatches to:
   - **[pattern-collector-routesjs-import](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-import)**
   - **[pattern-collector-routesjs-use](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-use)**
3. When **[pattern-collector-routesjs-import](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-import)** updates and publishes, it dispatches to:
   - **[pattern-collector-routesjs-import-extract](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-import-extract)**
4. When **[pattern-collector-routesjs-use](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-use)** updates and publishes, it dispatches to:
   - **[pattern-collector-routesjs-use-extract](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-use-extract)**
5. Both **[pattern-collector-routesjs-import-extract](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-import-extract)** and **[pattern-collector-routesjs-use-extract](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-use-extract)** update and dispatch to:
   - **[pattern-collector-routesjs-pull-lines](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-pull-lines)**
6. When **[pattern-collector-routesjs-pull-lines](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-pull-lines)** updates and publishes, it dispatches to:
   - **[pattern-collector-routesjs](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs)** (the final aggregate package).

### Chain B: Base Regex Cascade
1. **[pattern-collector-base-regex](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-base-regex)** is published.
2. Triggers dispatch to:
   - **[pattern-collector-routesjs-import-extract](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs-import-extract)**
3. Then follows Chain A steps 4-6 to update down to **[pattern-collector-routesjs](file:///d:/KeshavSoftRepos/2026-07-22(5)/pattern-collector-routesjs)**.
